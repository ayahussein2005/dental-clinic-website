# Dental Clinic Website 🦷

A professional dental clinic website built with HTML, CSS, JavaScript and Bootstrap 5.

## Pages
- **Home** - Main page with clinic overview
- **Services** - Available dental services
- **Doctors** - Our dental team
- **Contact** - Appointment booking & contact form

## Technologies Used
- HTML5 / CSS3 / JavaScript
- Bootstrap 5
- Font Awesome Icons
- Responsive Design

## Features
- Fully responsive design for all devices
- Modern and clean UI
- Appointment booking form
- Professional medical design
